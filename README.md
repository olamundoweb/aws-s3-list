# 🗂️ aws-s3-list

GitHub Action para listar arquivos de um bucket Amazon S3 utilizando a AWS SDK v3.

GitHub Action to list files from an Amazon S3 bucket using AWS SDK v3.

---

## 📄 Descrição / Description

Esta Action permite listar objetos armazenados em um bucket do Amazon S3 diretamente em workflows do GitHub Actions.

This Action allows you to list objects stored in an Amazon S3 bucket directly within GitHub Actions workflows.
